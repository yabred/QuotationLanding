# Quotation landing

Одностраничный лэндинг с цитатами: тёмная тема, карточки появляются при прокрутке (Intersection Observer).

## Локальный просмотр

Откройте `index.html` в браузере или поднимите статический сервер:

```bash
python3 -m http.server 8080
```

Затем откройте http://localhost:8080

## GitHub Pages

1. Запушьте репозиторий на GitHub.
2. В репозитории: **Settings → Pages**.
3. **Build and deployment → Branch:** выберите `main`, папка **/ (root)**.
4. Через минуту сайт будет доступен по адресу вида `https://<user>.github.io/<repo>/`.

Если репозиторий называется `<username>.github.io`, корень — это ваш пользовательский сайт.

## Файлы

| Файл | Назначение |
|------|------------|
| `index.html` | Разметка и тексты цитат |
| `styles.css` | Оформление и анимация карточек |
| `app.js` | Появление карточек при скролле |
